# ร้านน้ำ POS

POS ร้านน้ำเล็กสำหรับใช้งานส่วนตัว

## Main Features

* ขายเครื่องดื่ม
* เลือก size
* เพิ่ม topping
* ตะกร้า
* คิดเงิน
* ยอดขายวันนี้
* จัดการเมนู

## Devices

* iPad landscape (main)
* mobile portrait

## Tech Stack

* Next.js
* TailwindCSS
* TypeScript
* Google Apps Script
* Google Sheets

## UI Style

* minimal cafe
* cream + coffee tone
* large touch-friendly buttons

## Development Philosophy

* simple first
* usable first
* avoid overengineering
* build fast
* easy maintenance

## Current Phase

MVP Phase
