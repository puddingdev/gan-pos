# HARD RULES

* Do not add unnecessary dependencies
* Do not create enterprise architecture
* Do not create complex backend systems
* Do not add authentication
* Do not add inventory system
* Do not add analytics dashboard
* Do not add websocket
* Do not add real-time sync
* Do not add Docker
* Do not add Prisma

IMPORTANT:
This is a lightweight local cafe POS.

Everything should stay:

* simple
* maintainable
* deployable
* mobile friendly

When unsure:
choose the simpler solution.
